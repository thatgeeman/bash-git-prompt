## bash-git-prompt
Modify PS1 to show git branch in a bash terminal.

## How to
- update path to git-prompt.sh in `bashrc` 
- copy the contents of `bashrc` to your `~/.bashrc`.
- replace any other instances of `PS1` in `~/.bashrc` with `$PS1`.
